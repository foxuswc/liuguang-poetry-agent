# 流光诗境智能体网站

本网站是“流光诗境：基于大语言模型的情感驱动型诗歌教学智能体设计与应用研究”的结项演示成果。当前版本已接入统编版义务教育语文教材古诗文资源库，并提供可选高清图像生成接口。

## 打开方式

方式一：直接双击 `index.html` 即可在浏览器中打开。此模式不需要安装依赖或配置密钥，可完成情绪输入、诗歌推荐、资源库检索和提示词生成；右侧图像区在未生成真实图片前保持空白。

方式二：如需使用高清图像生成，请先在本文件夹的 `.env.local` 中填写 `OPENAI_API_KEY`。如果使用 OpenAI-compatible 转发接口，还需要填写 `OPENAI_BASE_URL`，例如：

```bash
OPENAI_BASE_URL=https://ai.input.im/v1
```

然后运行：

```bash
npm start
```

然后访问 `http://127.0.0.1:8765/`。如果用 `PORT=8766 npm start` 启动，则访问 `http://127.0.0.1:8766/`。接口通过本地 `server.js` 代理调用图像模型，浏览器端不会接触 API key。默认优先请求 `/images/generations`；若第三方兼容接口返回上游不可用，系统会自动改用 `/responses` + `image_generation` 工具。可通过 `OPENAI_BASE_URL`、`OPENAI_IMAGE_MODEL`、`OPENAI_RESPONSES_IMAGE_MODEL`、`OPENAI_IMAGE_SIZE`、`OPENAI_IMAGE_QUALITY`、`OPENAI_IMAGE_FORMAT` 调整。

如果页面提示 `Billing hard limit has been reached` 或“账单/预算上限已达到”，说明真实高清图没有生成。需要在 OpenAI Platform 中提高项目预算、切换有额度的 Project，或更换可用密钥后重启本地服务。

## 演示路径

1. 进入“学生交互”视图。
2. 在文本框中输入一段当下心情，或点击示例输入。
3. 调整“能量值”和“愉悦度”滑块，或直接点击二维情绪坐标。
4. 点击“生成诗歌引导”。
5. 查看系统推荐的诗歌、共情引导、推荐理由和视觉意象提示词。
6. 在已启动本地服务并配置密钥的情况下，点击“生成高清意象图”；未配置或生成失败时，右侧图像区保持空白，仅保留提示词与状态说明。
7. 点击“保存本次案例”，可在“资源库”视图查看本地案例记录。
8. 在“资源库”视图中搜索诗名、作者或情绪标签，也可导出 JSON 数据。

## 功能对应

- 情绪感知：文本关键词识别 + 罗素情绪环二维坐标。
- 文学匹配：按情绪坐标距离、标签命中和文本线索加权推荐诗歌。
- 资源库：覆盖统编版义务教育语文教材古诗文条目，并标注年级、册别、文本类型、情绪坐标和教学场景。
- 视觉升华：根据诗歌视觉意象生成文生图提示词，并可通过后端代理调用高质量生图模型。
- 教学应用：支持教师用于诗歌导入、情绪表达、读写结合和美育讨论。

## 文件结构

- `index.html`：智能体主页面。
- `styles.css`：界面样式。
- `app.js`：情绪识别、诗歌匹配、资源库、历史记录等交互逻辑。
- `data/poems.js`：教材诗歌资源库数据。
- `server.js`：可选本地服务与高清图像生成代理。
- `package.json`：本地服务启动脚本。

## 说明

本版本为结项增强版，采用本地规则、结构化教材数据库与可选图像模型接口模拟大语言模型应用流程。项目成果中的模拟试点数据仅用于展示研究设计和分析流程，不应替代真实课堂实证结论。
